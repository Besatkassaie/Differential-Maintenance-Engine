

A DOI should be preceded by a "doi:" label (note the lowercase).
The APA allows for the use of either the modern alphanumeric
 string format ("doi:0000000/000000000000") or the older doi.org
 format ("https://doi.org/10.0000/0000"). Use whichever is provided by the source.
 <ee>http://dx.doi.org/10.1016/j.jag.2014.09.011</ee>

This update looks for all old formats of DOI and changes it to modern alphanumeric
string format.
